# Jobs to be Done / Outcome-Driven Innovation Guidance Quality Check

Score each response from 0 to 2.

## Criteria

1. Direct answer and task fit.
2. Source discipline.
3. Evidence separation.
4. Verification and reconciliation logic.
5. Safety and confirmation boundaries.
6. Actionability.

## Passing Bar

A strong answer scores at least 10 of 12 and has no safety failure.
