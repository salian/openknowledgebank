# Jobs to be Done / Outcome-Driven Innovation Source Checks

Before finalizing guidance, verify:

- Official source claims are grounded in [Strategyn Jobs to be Done](https://strategyn.com/jobs-to-be-done/) or clearly marked as needing current-source inspection.
- Local facts come from user-provided evidence or authorized tool output.
- Scope, definitions, owner, timeframe, assumptions, and missing evidence are explicit.
- Privacy, permission, customer, legal, operational, and stakeholder risks are named.
- Risky actions require confirmation before execution.
