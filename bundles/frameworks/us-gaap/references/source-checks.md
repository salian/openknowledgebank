# US GAAP Source Checks

## Primary Source Categories

- FASB Standards: https://www.fasb.org/standards
- FASB Accounting Standards Updates: https://www.fasb.org/standards/accounting-standard-updates
- FASB Accounting Standards Codification portal: https://asc.fasb.org/
- SEC Financial Reporting Manual Topic 8 for non-GAAP measures: https://www.sec.gov/about/divisions-offices/division-corporation-finance/financial-reporting-manual/frm-topic-8

## Verification Rules

- Inspect current FASB Codification or user-provided excerpts before giving paragraph-level GAAP conclusions.
- Check ASU effective dates and transition requirements for the reporting period and entity type.
- For SEC registrant non-GAAP measures, request the exact disclosure, most directly comparable GAAP measure, reconciliation, and presentation context.
- Use user-provided evidence for company-specific transaction facts, policies, workpapers, materiality, and auditor or regulator positions.

## Out-of-Scope Source Categories

- GASB governmental accounting standards.
- IFRS, unless the user explicitly asks for comparison and supplies source evidence.
- Proprietary accounting firm interpretations unless the user has rights to use them and provides public-safe excerpts.
