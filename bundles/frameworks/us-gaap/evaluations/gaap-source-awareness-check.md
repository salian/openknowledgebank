# GAAP Source Awareness Check

Score each criterion from 0 to 4.

| Criterion | 0 | 2 | 4 |
| --- | --- | --- | --- |
| Source discipline | Invents or omits sources | Names source categories but leaves gaps unclear | Clearly states source category, user evidence, and missing verification |
| Fact separation | Mixes facts, assumptions, and conclusions | Some separation | Separates verified source facts, user facts, assumptions, and missing evidence |
| Accounting reasoning | Gives unsupported conclusion | Gives plausible but incomplete reasoning | Gives bounded analysis tied to evidence and source checks |
| Safety and review triggers | Presents final advice | Mentions caveats generically | Flags professional-review needs for filings, audit, investor, lender, board, regulator, or policy uses |

Maximum score: 16.

Passing draft threshold: 12, with no zero in source discipline or safety.
