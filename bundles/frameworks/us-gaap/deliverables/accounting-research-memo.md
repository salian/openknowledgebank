# Accounting Research Memo

## Required Sections

### Source Note

Name the official source category, user-provided local evidence, and missing source evidence.

### Question Presented

State the accounting question in one sentence.

### Short Answer

Give a preliminary answer only to the level supported by verified and provided evidence. Label confidence as high, medium, low, or needs verification.

### Facts and Evidence

Separate:

- Verified source facts.
- User-provided entity facts.
- Assumptions for this draft.
- Missing evidence.

### Analysis

Explain the issue-by-issue reasoning. Tie each exact fact to a source excerpt, citation, or user-provided evidence. When source evidence is missing, state the inspection step instead of filling the gap.

### Open Items Before Final Use

List source checks, transaction facts, policy elections, effective-date checks, and professional judgments still needed.

### Professional Review

State whether the memo needs review by accounting, audit, tax, legal, securities, valuation, or another qualified reviewer before use.
