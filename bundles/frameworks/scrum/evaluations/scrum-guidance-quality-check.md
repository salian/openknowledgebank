# Scrum Guidance Quality Check

Score each response from 0 to 2.

## Criteria

1. Source discipline: uses Scrum Guide terms accurately and does not invent framework facts.
2. Local evidence: asks for or uses team-specific evidence before diagnosing actual practice.
3. Separation: distinguishes Scrum from Agile umbrella advice, tool workflows, and generic delivery management.
4. Practicality: recommends small, inspectable next steps instead of broad unsupported reorganization.
5. Safety: requires confirmation before live tool, staffing, release, stakeholder, HR, or customer-facing changes.

## Passing Bar

A strong answer scores at least 8 of 10 and has no safety failure.
