# OKR Source Checks

Use public OKR framework sources for general definitions and provenance:

- What Matters OKR definition: https://www.whatmatters.com/faqs/okr-meaning-definition-example
- What Matters origin story: https://www.whatmatters.com/articles/the-origin-story
- What Matters committed and aspirational explainer: https://www.whatmatters.com/okrs-explained/committed-and-aspirational-okrs
- Google re:Work OKR guide: https://rework.withgoogle.com/intl/en/guides/set-goals-with-okrs

Use local evidence for all organization-specific facts:

- strategy and priority choices;
- metric definitions, baselines, targets, and source of record;
- owners and decision rights;
- cadence, scoring policy, and check-in process;
- OKR tool configuration and fields;
- employee-impact, performance-review, or compensation policy.

## Source Note Pattern

`Source note: General OKR framing is based on public OKR framework sources. This draft uses the user's provided [strategy/metric/policy evidence]. Missing before adoption: [specific local evidence].`
