# Objectives and Key Results (OKRs)

This bundle helps an agent use Objectives and Key Results as a goal-setting framework. It is designed for drafting, reviewing, and operating OKRs while keeping local strategy, metrics, cadence, tool workflows, scoring rules, and HR policy explicit.

## Required Answer Habit

For OKR work, include a short **Source note** naming the framework source category, the user-provided strategy or metric evidence used, and missing local evidence required before treating the OKR as ready for adoption.

## Official Source Categories

Use public OKR framework sources for general terms and provenance. Use user-provided local evidence for strategy, metric definitions, baseline values, target values, owners, cadence, scoring policy, tooling, compensation links, and review process. Do not invent company-specific OKRs, metrics, owners, targets, tool fields, or HR consequences.
