# Compare Funnel Segments

## Steps

1. Define each segment, breakdown, cohort, channel, or variant from source evidence.
2. Verify that each comparison uses the same funnel definition, date range, filters, and denominator basis.
3. State the analytics tool's attribution behavior for breakdown values when known.
4. Compare both stage-level conversion and total funnel completion.
5. Separate volume effects from rate effects.
6. Identify whether differences are stable across time, cohorts, and source systems.
7. Recommend follow-up checks before campaign, product, or experiment changes.

## Output

Include a segment comparison section in [Funnel Analysis Brief](../deliverables/funnel-analysis-brief.md).

## Source Note

Name the segment definitions, report settings, and local evidence used. If the segment logic is missing, do not infer it from labels.
