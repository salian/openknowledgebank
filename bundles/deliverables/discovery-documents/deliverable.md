## Value Signal

Discovery documents (requests for production, interrogatories, privilege logs, deposition notices) are a highly structured, procedurally regulated litigation artifact class — governed by court rules (FRCP or state equivalents), with strict formatting, privilege assertions, and objection conventions. The base model knows the general concept but lacks the disciplined structure, boilerplate patterns, and defensibility checklist (e.g., privilege log required elements: date, author, recipients, privilege basis, document description without waiving privilege) that a real paralegal workflow demands. This makes it a good candidate for a nuanced, checklist-style bundle contribution despite being single-role.
