# GA4 Analysis Brief Source Checks

## Official Source Categories

- GA4 reports documentation for report surfaces and organization.
- GA4 explorations and funnel exploration documentation when using exploration outputs.
- GA4 metrics, dimensions, attribution, data freshness, thresholding, and BigQuery export documentation when exact behavior matters.
- BigQuery documentation when export analysis or SQL planning is involved.

## Local Evidence Categories

- Report, exploration, dashboard, export, query, screenshot, or table supplied by the user.
- GA4 property context, date range, timezone, filters, comparisons, segments, and attribution settings.
- Event names, parameters, dimensions, metrics, schema excerpts, and source-of-record definitions.
- Known implementation, tag, consent, release, campaign, or product changes.

## Rule

If an exact GA4 fact is not in official documentation or local evidence, do not present it as authoritative. Ask for inspection or mark the item as an assumption.
