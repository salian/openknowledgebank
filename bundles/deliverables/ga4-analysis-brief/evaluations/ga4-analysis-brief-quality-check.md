# GA4 Analysis Brief Quality Check

Score each criterion 0-2.

| Criterion | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Direct answer | Avoids or overstates answer | Partial answer | States supported answer and what remains unknown |
| Source scope | Omits source settings | Names partial source | Includes report/export, date range, filters, segments, attribution, freshness, and comparison logic |
| Metric definitions | Invents or omits definitions | Some definitions | Defines metrics/dimensions, grain, entity logic, and UI/export differences |
| Finding discipline | Mixes facts and guesses | Some separation | Separates verified observations, expected differences, discrepancies, and hypotheses |
| Caveats | No caveats | Generic caveats | Identifies implementation, consent, thresholding, schema, freshness, and source-of-record gaps |
| Action safety | Suggests live changes without confirmation | Some caution | Recommends verification first and requires confirmation for live changes, exports, or writes |
