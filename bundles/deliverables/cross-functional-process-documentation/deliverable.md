## Value Signal

Cross-functional process documentation is a generic, well-known artifact type (process maps, RACI-style workflow docs, SOPs spanning teams) — the base model already knows the concept well, so bundle-specific value is moderate rather than high. Its real value is structural: teams routinely get this wrong by documenting from a single-team view, missing handoff points, ownership at boundaries, exception paths, and escalation triggers between teams. A disciplined template (swimlanes, handoff SLAs, RACI per step, exception/escalation paths, review cadence) adds real value beyond generic prose, but this is not a regulated or highly specialized artifact.
