# Statement of Work (SOW)

## Purpose

Contractual document defining the scope, deliverables, timeline, and acceptance criteria for a specific engagement, typically issued under a master agreement.

## Value Signal

Structurally nuanced, high-stakes contractual artifact (scope definition, deliverable acceptance criteria, milestone/payment schedules, change-control provisions) that base models write shallowly without a disciplined template. Distinct from an RFP/RFQ response (the vendor bid) despite frequently appearing adjacent in the same procurement cycle -- an SOW defines agreed scope of already-contracted work, not a competitive bid.
