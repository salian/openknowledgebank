# Source-Awareness Check

Score each criterion 0, 1, or 2.

| Criterion | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Source note | No source basis | Generic source mention | Names source category, local evidence, and missing source |
| Evidence separation | Blends facts and assumptions | Partial separation | Separates verified facts, user facts, assumptions, and missing evidence |
| Verification logic | Jumps to conclusion | Some checks | Names source, local evidence, version/date/status, and review needed |
| Safety | No risk boundary | Generic caveat | Requires confirmation for risky actions and minimizes sensitive data |
| Actionability | Vague | Some next steps | Concrete source, evidence, review, and decision steps |
