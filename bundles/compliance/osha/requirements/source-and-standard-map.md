# OSHA Source and Standard Map

Use this map to route an OSHA question before drafting a conclusion.

| Question Type | Primary Source Category | User Evidence Needed | Drafting Rule |
| --- | --- | --- | --- |
| General OSHA duty or employer responsibility | OSHA laws-regs and employer-responsibilities pages | Employer, workplace, hazard, and action facts | State high-level responsibility only; inspect current sources for exact duties |
| Worker right or complaint issue | OSHA worker-rights pages and relevant standards | Worker role, alleged hazard, timing, communications, and jurisdiction | Do not give retaliation or employment-law strategy without professional review |
| Federal standard applicability | eCFR Title 29 Chapter XVII and OSHA standards pages | Industry, establishment, process, task, equipment, material, hazard, and state-plan facts | Identify candidate standard family; do not declare applicability without current source review |
| General industry standard | 29 CFR Part 1910 | Worksite and task facts | Use as source family; inspect relevant subpart/section before exact answer |
| Construction standard | 29 CFR Part 1926 | Construction activity, site role, contract, task, hazard, and state-plan facts | Use as source family; inspect relevant subpart/section before exact answer |
| Recordkeeping or reporting | OSHA recordkeeping resources and 29 CFR Part 1904 | Employer size/industry, establishment, incident facts, medical treatment, days away/restriction, outcome, timing | Do not decide recordability/reportability from incomplete facts |
| Poster requirement | OSHA poster page | Workplace display facts, workforce language needs, state-plan context | Check current OSHA/state source before relying on details |
| Hazard communication | OSHA HazCom page and applicable 29 CFR source | Chemical inventory, labels, SDSs, training, workplace program, industry/state-plan facts | Treat as source category; do not invent SDS, label, or training requirements |
| State-plan issue | OSHA state-plan sources and state authority pages | State, industry, employer, public/private employer status, worksite facts | State-plan source inspection is mandatory before conclusion |
| Inspection, citation, or penalty question | OSHA enforcement/citation sources and user documents | Citation, inspection, notice, dates, abatement, correspondence, counsel status | Require legal/professional review and do not infer deadlines or penalty exposure |

## Source Note Pattern

Every answer should say which official source category was inspected, which user evidence was provided, which assumptions are being made, and which missing sources or facts block reliance.
