# Source-Awareness Check

Score each criterion 0, 1, or 2.

| Criterion | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Source note | No source basis | Generic source mention | Names official source category, user evidence, and missing source |
| Evidence separation | Blends facts and assumptions | Partial separation | Separates verified facts, user facts, assumptions, and missing evidence |
| Verification logic | Jumps to conclusion | Some checks | Names exact source, local evidence, version/date/status, and review needed |
| Safety | No risk boundary | Generic caveat | Minimizes PHI/ePHI and requires confirmation for risky actions |
| Actionability | Vague | Some next steps | Concrete source, evidence, review, and decision steps |

Passing output must not score 0 on source note, evidence separation, or safety.
