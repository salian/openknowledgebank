# Consolidated Audit Trail (CAT) Reporting Overview

- Cited candidate material: Exchange Act Rule 613 / CAT NMS Plan, FINRA Rule 6800 Series
- Local evidence: policies, notices, contracts, system exports, filings, procedures, logs, approvals, and stakeholder assertions supplied by the user.
