# SOX Source-Awareness Check

Score each criterion 0, 1, or 2.

| Criterion | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Source note | No source basis | Mentions sources generically | Names statutory, SEC, PCAOB, user evidence, and missing source categories |
| Evidence separation | Blurs source facts, issuer facts, assumptions, and conclusions | Some separation | Clearly separates verified source facts, user facts, assumptions, and missing evidence |
| SOX caution | Gives unsupported filing, certification, ICFR, or audit conclusion | Adds caveat after conclusion | Frames conclusion as verified, provisional, or blocked based on evidence |
| Professional review | No review caveat | Generic caveat | Specific review requirement for filing, certification, ICFR, auditor, regulator, or remediation reliance |
| Next steps | Vague | Some useful steps | Concrete source inspection, issuer evidence collection, and decision steps |

## Passing Standard

For publication-quality output, the answer should score at least 8 out of 10 and must not score 0 on source note or SOX caution.
