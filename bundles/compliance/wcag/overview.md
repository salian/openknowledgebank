# Overview

WCAG is the common shorthand for Web Content Accessibility Guidelines. This bundle treats W3C/WAI as the primary standards source, user-provided product evidence as necessary for product-specific conclusions, and legal, procurement, contractual, or jurisdictional materials as separate source categories that must be inspected when relevant.

## What This Bundle Is For

- Triage whether a question is about WCAG version, conformance level, product scope, issue remediation, testing evidence, public accessibility statements, VPAT/ACR material, or legal/procurement obligations.
- Identify the official source categories that must be checked before answering.
- Produce source-aware accessibility briefs that separate verified WCAG facts from user evidence, assumptions, and missing verification.
- Route deeper success-criterion, jurisdiction, scanner, or audit questions to current official sources and qualified review.

## What This Bundle Is Not For

- It is not legal advice.
- It is not a WCAG certification or accessibility audit.
- It does not decide product conformance from incomplete scanner output.
- It does not replace current W3C/WAI source inspection, manual accessibility testing, assistive-technology review, jurisdiction-specific legal review, procurement review, or professional accessibility judgment.

## Source Discipline

Every WCAG output should distinguish:

- verified W3C/WAI standard facts;
- exact success-criterion wording or conformance language inspected for the specific task;
- facts provided by the user about product scope, code, content, testing, audit, or remediation;
- assumptions made only for a draft;
- missing facts or sources that block a conformance, legal, procurement, or public-claim conclusion.

## Risk Boundary

Publishing accessibility statements, submitting VPAT/ACR materials, claiming conformance, changing live products, contacting regulators, signing procurement responses, or sending legal/compliance communications are risky actions. An agent should require explicit user confirmation and qualified professional review before those actions.
