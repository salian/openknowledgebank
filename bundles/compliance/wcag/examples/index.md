# Examples

## Illustrative Before/After

Task: A product manager asks whether a site is WCAG 2.2 AA compliant based on a small scanner report.

Baseline-style answer: "The scanner found only a few issues, so the site is probably close to WCAG 2.2 AA compliant after fixing those."

Bundle-assisted answer: "The scanner report is evidence but not a complete conformance determination. Source note: W3C/WAI WCAG 2.2 AA source wording and conformance requirements still need inspection for the cited criteria; user evidence includes the scanner report only; missing evidence includes page/component scope, manual keyboard testing, assistive-technology checks, remediation evidence, and any legal/procurement source. The conclusion is provisional: fix and verify the listed issues, define the conformance scope, run manual and assistive-technology review where appropriate, and route any public conformance statement or VPAT/ACR claim to qualified review."
