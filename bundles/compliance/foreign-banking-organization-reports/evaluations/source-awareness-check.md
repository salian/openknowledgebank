# Foreign Banking Organization Reports (FR Y-7 series) Source-Awareness Check

A passing answer should:

- Name the source category needed for the claim.
- Avoid inventing private facts, current source text, or system access.
- Separate verified evidence from assumptions and missing facts.
- Include a qualified-review boundary for final reliance.
