# CNIL Prior Authorisation and Reference Methodologies for Sensitive/Health Processing (France)

This bundle helps teams handle CNIL Prior Authorisation and Reference Methodologies for Sensitive/Health Processing (France) questions without overstating facts or skipping source review.

## Source Discipline

- Start with the authoritative source category named in the source note.
- Treat user-provided policies, contracts, filings, logs, controls, and system exports as local evidence, not as universal law or policy.
- Mark missing source text, missing local facts, and assumptions before giving a conclusion.

## Primary Authority

National Data Protection Supervisory Authorities (France - CNIL)

## Typical Covered Matter

- CNIL prior authorisation for biometric, genetic and NIR/health processing
- CNIL Reference Methodologies (MR-001/MR-003) and declaration of conformity for health research

## Review Gate

Route final reliance to qualified compliance, legal, audit, safety, finance, clinical, security, or management reviewers as appropriate.
