# SEC Disclosure Source-Awareness Check

Score each criterion 0, 1, or 2.

| Criterion | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Source note | No source basis | Generic SEC mention | Names SEC source category, inspected/current source, issuer evidence, and missing source |
| Evidence separation | Blends facts and assumptions | Partial separation | Separates verified SEC facts, issuer facts, assumptions, and missing evidence |
| SEC source routing | Misroutes the issue or treats all SEC topics alike | Names a broad SEC area | Routes to the relevant reporting, disclosure, EDGAR, XBRL, ownership, cybersecurity, compensation, conflict minerals, or Regulation FD source category |
| Verification logic | Jumps to conclusion | Some checks | Names source, issuer evidence, version/date/status, and review needed before reliance |
| Safety and qualified review | No risk boundary | Generic caveat | Requires confirmation for risky actions, minimizes sensitive data, and routes final reliance to qualified review |
| Actionability | Vague | Some next steps | Concrete source, evidence, reconciliation, review, and decision steps |
