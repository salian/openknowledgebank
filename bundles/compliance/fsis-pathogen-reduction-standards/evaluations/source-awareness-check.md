# Pathogen Reduction Performance Standards and Process-Control Testing (9 CFR 310.25; Directive 10,250.1) Source-Awareness Check

Check that responses name the source category, separate local evidence from source facts, avoid invented access, list missing evidence, and route final reliance to qualified review.
