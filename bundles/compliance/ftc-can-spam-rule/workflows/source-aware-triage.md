# CAN-SPAM Rule, 16 CFR Part 316 (implementing the CAN-SPAM Act) Source-Aware Triage

1. Identify the scenario, jurisdiction, regulated entity, product, workflow, transaction, filing, record, or population at issue.
2. Name the official source category and inspect current source text before relying on specific obligations.
3. Separate verified source facts from local evidence and assumptions.
4. Ask for missing policies, contracts, filings, logs, controls, exports, notices, approvals, or reviewer decisions.
5. Produce a short triage answer with a confidence note and qualified-review gate.

## Output Format

- Source note
- Known facts
- Missing evidence
- Applicability analysis
- Risks and caveats
- Next checks
- Qualified reviewer
