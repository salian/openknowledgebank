# Audit Firm Governance Code

This bundle helps teams handle Audit Firm Governance Code questions without overstating facts or skipping source review.

## Source Discipline

- Start with the authoritative source category named in the source note.
- Treat user-provided policies, contracts, filings, logs, controls, and system exports as local evidence, not as universal law or policy.
- Mark missing source text, missing local facts, and assumptions before giving a conclusion.

## Primary Authority

Financial Reporting Council (FRC)

## Typical Covered Matter

- Audit Firm Governance Code

## Review Gate

Route final reliance to qualified compliance, legal, audit, safety, finance, clinical, security, or management reviewers as appropriate.
