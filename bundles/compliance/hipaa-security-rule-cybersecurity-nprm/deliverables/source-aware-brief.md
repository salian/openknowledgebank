# Nprm Status And Impact Brief

## Direct Answer

State what can be said now and whether the answer is verified, provisional, or blocked by missing evidence.

## Source Note

Name the official source category, current source inspected or still needed, user-provided evidence, and missing source or fact.

## Evidence Matrix

| Claim | Status | Source or Evidence | Caveat |
| --- | --- | --- | --- |
| Official requirement or standard | verified / needs verification | HHS/OCR, CMS, eCFR, Federal Register, implementation guide, or missing source | Cite only inspected sources. |
| Local organization fact | provided / missing | User evidence | Do not infer facts. |
| Draft assumption | assumed | Stated assumption | Do not treat as conclusion. |
| Operational action | needs review | Professional review and explicit confirmation | Do not act without confirmation. |

## Analysis

Connect the verified source category to the provided local facts. If the answer depends on facts not supplied, say so directly.

## Next Steps

- Inspect the named official source.
- Collect minimum necessary, redacted local evidence.
- Route to qualified professional review.
- Require explicit confirmation before any live or external action.
