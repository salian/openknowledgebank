# Color Additive Batch Certification (21 CFR Parts 70, 74, 80) Source-Aware Triage

1. Restate the user question and identify the decision they want to make.
2. Name the jurisdiction, authority, source category, and local evidence currently available.
3. Separate verified source facts from user-provided facts, assumptions, and missing evidence.
4. Check applicability, scope, exemptions, timing, required actions, records, notices, reporting, and review ownership only against inspected sources.
5. Provide a cautious answer with explicit open questions and a qualified-review gate before reliance.
