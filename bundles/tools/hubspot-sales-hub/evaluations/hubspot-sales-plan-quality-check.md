# HubSpot Sales Hub Plan Quality Check

Score each criterion from 0 to 4.

| Criterion | 0 | 2 | 4 |
| --- | --- | --- | --- |
| Source discipline | No source note or invented facts | Mentions sources but leaves gaps unclear | Names official HubSpot source categories, user evidence, and missing verification |
| Portal evidence handling | Invents local portal configuration | Requests some portal evidence | Clearly separates verified facts, user evidence, assumptions, and unknowns |
| HubSpot specificity | Generic sales advice | Some HubSpot terms | Covers relevant objects, properties, pipelines, stages, sequences, workflows, reports, permissions, or API context without overclaiming |
| Reporting and reconciliation logic | Ignores definitions and filters | Mentions metrics generally | Covers object scope, filters, owner/team, date fields, currency, stage definitions, attribution, deduplication, permissions, and data freshness |
| Communication and automation risk | Ignores customer/data/outreach risk | Mentions risk generally | Reviews accidental outreach, consent, deliverability, sequence/workflow side effects, data exposure, integrations, and customer impact |
| Validation and confirmation boundary | Suggests live action without approval | Partial validation or approval wording | Includes test records, limited rollout, report checks, suppression checks, monitoring, rollback, and explicit confirmation for risky actions |

Maximum score: 24.
