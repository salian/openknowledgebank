## Value Signal

Factorial is an SMB-focused all-in-one HR/HRIS platform (core HR, time tracking/attendance, leave/absence, payroll, finance integrations) with strong adoption in Europe and Latin America rather than global ubiquity — a regional/vertical category player closer to TOPdesk than to a horizontally-ubiquitous P0 platform. It exposes a genuine proprietary REST API (OAuth 2.0 authorization-code flow with token refresh/revocation, API keys, partner/company tokens, webhooks, pagination, separate production/demo environments) and date-based API versioning (v2025-07-01 through v2026-07-01), plus product-specific data models (employees, shifts/clock-in, time-off/leave, payroll, finance). Base models have general HR-software awareness but are weak on Factorial's specific endpoint shapes, versioned/fast-changing API surface, and object schema, so a grounded bundle adds real per-fact value for HR-ops, people-ops, payroll-admin, and SMB-founder workflows. Hub/SEO reach is moderate (HR/payroll touches several role bundles) but Factorial's regional, SMB-scoped installed base keeps it below the cross-role ubiquity of global HR leaders (Workday, BambooHR, Gusto, Rippling), so P2 rather than P1.
