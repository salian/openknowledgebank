## Value Signal

SAP Concur is the dominant enterprise travel, expense, and invoice spend-management platform, broadly adopted by large organizations and deeply integrated (SAP/ERP, NetSuite, travel suppliers, corporate card networks, App Center marketplace). It carries strong hub/SEO value as the category leader, and many role bundles (finance/AP, accounting, FP&A, controllers, procurement, travel managers, finance-systems admins) would plausibly link to it. Per-bundle grounding value is genuinely high: the official developer reference exposes a proprietary, versioned, fast-changing REST API surface (OAuth 2.0 scopes, Expense v1/v3/v4, Travel/Itinerary, Invoice & Procurement, Request, Budget, Identity/User provisioning, Cards, receipt/compliance gateway) plus idiosyncratic policy and workflow semantics (expense report lifecycle, allocations, travel allowance calc, audit rules, callout triggers) that base models know only superficially. Not P0 — it is a specialized finance/T&E vertical platform rather than a horizontal-ubiquitous tool like Salesforce/Snowflake/Jira — so it calibrates to P1 alongside other leading enterprise platforms (Marketo, Workfront).
