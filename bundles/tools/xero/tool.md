## Value Signal

One of the two dominant global cloud accounting platforms (alongside Intuit QuickBooks), market-leading in ANZ and the UK with millions of subscribers, a large app marketplace (1000+ integrations), and unlimited users. Base model is reasonably strong on Xero's core domain (invoicing, bank reconciliation, chart of accounts, contacts) so per-bundle informational value is moderate — but it carries very high hub/SEO value as a default accounting pick across finance, bookkeeping, accountant, controller/FP&A, ops, and SMB-founder role bundles. The marginal grounding value sits in the proprietary, fast-changing Accounting REST API: object model (Invoices, Contacts, Accounts, BankTransactions, TrackingCategories, Journals), OAuth2/PKCE flow with multi-tenant connections, scopes, rate limits, and region-specific tax/payroll surfaces — areas the base model covers only shallowly.
