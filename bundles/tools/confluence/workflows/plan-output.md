# Confluence Output Planning

1. Define the output type and success criteria.
2. Confirm scope, owner, timeframe, definitions, and constraints.
3. Map source facts to local evidence.
4. Draft the smallest useful artifact.
5. Add assumptions, missing evidence, and review steps.
6. Mark any live action or external communication as requiring confirmation.
