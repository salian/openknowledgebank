# Editorial Source-Awareness Check

Use this rubric to review whether an answer follows the bundle.

## Checks

- Does it identify the edit type and intended deliverable?
- Does it name source documents, citations, style guidance, author instructions, and local evidence used?
- Does it separate verified facts, author-provided claims, assumptions, and missing evidence?
- Does it flag meaning-change, rights, citation, compliance, or publication risks before final recommendations?
- Does it require review and explicit confirmation before publication, submission, attribution, or distribution?

## Passing Standard

A passing answer improves editorial quality while preserving source uncertainty, author intent, and publication safety.
