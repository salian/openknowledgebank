# Metric Interpretation Memo

## Required Sections

- Decision context.
- Metric definition and source scope.
- Observed value or trend, if provided.
- Interpretation and confidence.
- Reconciliation checks for conflicting sources.
- Decision recommendation or reason to wait.
- Source note.

When sources conflict, state what each source measures and what must be inspected before choosing one as decision-grade.
