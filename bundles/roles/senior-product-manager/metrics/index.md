# Metrics

Treat product metrics as definitions plus evidence, not as isolated numbers.

## Minimum Metric Context

- Decision the metric will inform.
- Metric definition, numerator, denominator, filters, segments, identity logic, aggregation, deduplication, and refresh cadence.
- Source system and owner.
- Time period and comparison period.
- Instrumentation or business-process changes that could affect interpretation.

## Reconciliation Rule

If two sources disagree, neither number is automatically right. Explain what each source measures, align definitions and settings, list expected differences, and separate unresolved discrepancies before making a roadmap decision.
