# Evidence to Decision

Use four evidence classes:

- Verified source facts: inspected artifacts, dashboards, queries, interviews, research notes, tickets, or records.
- User-provided evidence: pasted or described by the user but not independently inspected.
- Assumptions for this draft: plausible placeholders that must not be treated as facts.
- Missing verification: evidence needed before commitment.

For each recommendation, state which evidence class carries the decision and what new evidence would change the recommendation.
