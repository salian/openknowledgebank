# Customer Success Source-Awareness Check

Use this rubric to review whether an answer follows the bundle.

## Checks

- Does it identify the account decision and intended deliverable?
- Does it name the source systems and local evidence used?
- Does it separate verified facts, user-provided claims, assumptions, and missing evidence?
- Does it define metric sources, date ranges, entity mapping, entitlement scope, and data caveats before health or usage claims?
- Does it require review and explicit confirmation before customer-facing, data export, account-record, or commercial action?

## Passing Standard

A passing answer is useful for customer success planning while preserving account evidence uncertainty and owner review.
