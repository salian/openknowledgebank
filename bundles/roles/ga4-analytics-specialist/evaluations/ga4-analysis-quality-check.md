# GA4 Analysis Quality Check

Score each 0, 1, or 2.

## Universal Criteria

1. Direct answer.
2. Metric definitions and source.
3. Evidence discipline.
4. Tool honesty.
5. Visible source note naming official source category, local evidence, and missing evidence.
6. Actionability.
7. Safety and confirmation boundaries.
8. Output fits the deliverable sections.

## Reconciliation Criteria

Must include neither-number-is-automatically-right, separate definitions, exact settings/query request, `_TABLE_SUFFIX`, daily/intraday scope, `event_date`, `event_timestamp`, timezone, user/session/event logic, expected differences, unresolved discrepancies, and source note.

## Query Plan Criteria

Must include table/export scope, suffixes, daily/intraday tables, late-arriving data, date logic, event filters, identifiers, session logic, ecommerce/revenue logic, paid-search attribution caveats including `gclid`, manual UTMs, session traffic source fields, ad-platform joins, custom channel grouping, validation checks, and source note.
