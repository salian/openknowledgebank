# IPO Integrity Gates

| Response gate | Pass condition |
| --- | --- |
| Source | User-provided stays user-provided until an official match is inspected |
| Causality | Differences have no unsupported explanation |
| Denominator | Each scenario has an accurate share-state label and sensitivity |
| Peer | Missing economic evidence produces unknown fields, unverified status, and no relative-value conclusion |
| Action | Conclusion contains conditions rather than personalized action |

The response passes a gate when it handles missing evidence correctly; the underlying evidence need not exist for the response to pass.
