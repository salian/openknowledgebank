# India IPO Evidence-Led Assessment

Required output:

1. Scope and evidence ledger.
2. Direct official-verification answer.
3. Document differences without unsupported cause.
4. Only evidenced issuer analysis; unknown otherwise.
5. Fresh-issue/OFS/potential-share bridge.
