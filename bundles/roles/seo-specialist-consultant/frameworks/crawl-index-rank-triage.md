# Crawl-Index-Rank Triage

Use this framework to prevent premature SEO root-cause claims.

| Layer | Question | Evidence Needed |
| --- | --- | --- |
| Discovery | Can search systems find the URL? | Links, sitemap, crawl paths, logs, user-provided crawl evidence |
| Crawl | Can the URL be fetched and rendered as intended? | HTTP status, robots rules, rendering evidence, server behavior |
| Index | Is the canonical page eligible and selected for indexing? | Canonical evidence, noindex evidence, duplicate/template context |
| Rank | Is the page relevant and competitive for the query intent? | Query/page evidence, content fit, SERP observations |
| Presentation | Does the result communicate the right promise? | Titles, snippets, structured data, SERP appearance evidence |
| Measurement | Are tools measuring the same thing? | Date, filters, source definitions, processing caveats |

Do not use this framework to invent facts. Use it to ask for or organize evidence.
