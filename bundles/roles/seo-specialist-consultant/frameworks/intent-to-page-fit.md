# Intent-to-Page Fit

Evaluate whether a page satisfies the task implied by the query or topic.

| Element | Check |
| --- | --- |
| Query or topic | What is the user trying to learn, compare, buy, fix, or verify? |
| Page promise | Does the title, heading, and introduction make a clear promise? |
| Evidence | Does the page provide facts, examples, data, products, credentials, or steps that support the promise? |
| Completeness | What must be added, removed, clarified, or reorganized? |
| Conversion path | What next action should the user be able to take? |
| Measurement | What source will show whether the change helped? |

Use this framework to improve usefulness and clarity, not to stuff keywords.
