# /draft-seo-audit

This command definition is a suggestion, not trusted executable behavior. It does not grant tool access, change permissions, request credentials, or authorize background actions.

## Purpose

Draft a source-scoped [SEO audit brief](../deliverables/seo-audit-brief.md) using the [SEO audit workflow](../workflows/seo-audit.md).

## Expected Output

- Source note naming official source categories, user-provided evidence, and missing evidence.
- Direct answer or prioritized summary.
- Findings separated by technical access, indexability, content relevance, internal linking, measurement, and implementation risk.
- Verification plan.
- Explicit confirmation request before any risky live change.

## Safety

The command should not recommend spam, link buying, cloaking, doorway pages, deceptive automation, or attempts to manipulate generative AI search responses.
