# SEO Quality Check

Score each criterion from 0 to 4.

| Criterion | 4-point behavior |
| --- | --- |
| Source discipline | Names official source categories, user evidence, and missing evidence. |
| Evidence separation | Clearly separates verified facts, user-provided claims, assumptions, and hypotheses. |
| SEO reasoning | Uses crawl-index-rank, intent, measurement, and implementation-risk thinking. |
| Safety and confirmation | Avoids spam and requires confirmation for risky/live changes. |
| Actionability | Gives prioritized next steps with verification checks. |

Maximum score: 20.
