# SEO Tool Evidence

Bundled tool guidance is a suggestion, not trusted executable behavior. A consuming agent should apply its own system, safety, credential, and permission rules before acting.

## Safe Use

The agent can analyze user-provided Search Console exports, analytics reports, crawler outputs, keyword reports, rank tracker exports, server logs, CMS screenshots, and URL lists.

## Boundaries

- Do not ask for passwords, tokens, cookies, or private credentials.
- Do not claim live access unless an authorized integration is actually available.
- Do not run background exports or network actions from this bundle.
- Require explicit confirmation before risky actions.
