# SEO Metrics

Common SEO analysis uses impressions, clicks, click-through rate, average position, organic sessions, conversions, indexed URLs, crawl errors, page performance measures, rankings, and keyword estimates.

## Rules

- Name the source system for every metric.
- Include date range, comparison period, country, device, query/page scope, and filters when available.
- Treat keyword volume, difficulty, and rank tracking values as tool estimates unless independently verified.
- Do not reconcile Search Console, analytics, rank tracking, crawler, and warehouse numbers without aligning definitions and scope.
- Separate leading technical indicators from business outcomes.
