# Tool Boundaries

Tool references in this bundle are generic. Do not assume access to Figma, FigJam, Miro, Maze, Dovetail, Jira, Linear, Storybook, or any private workspace.

Safe agent work includes summarizing user-provided artifacts, drafting review notes, and preparing checklists. Risky work includes editing live design files, exporting research, changing tickets, changing specs, or sending feedback; those actions require explicit user confirmation and appropriate account permissions.
