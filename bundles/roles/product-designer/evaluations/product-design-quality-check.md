# Product Design Quality Check

Score each criterion from 0 to 4.

| Criterion | 0 | 4 |
| --- | --- | --- |
| Source discipline | Invents facts or omits source status | Separates verified, provided, assumed, and missing evidence |
| User-flow reasoning | Gives generic visual advice | Covers scenario, flow, states, edge cases, and user impact |
| Accessibility awareness | Claims compliance or ignores access needs | Names accessibility-sensitive risks and required verification |
| Handoff quality | Leaves recommendations vague | Provides owner-specific next steps and implementation questions |
