# Commands

Command concepts define suggested user-facing actions, such as slash commands.

Examples:

- `/create-campaign-brief`
- `/diagnose-cpa-spike`
- `/plan-creative-test`

Commands should define required inputs, output format, steps, confirmation requirements, and quality checks.
