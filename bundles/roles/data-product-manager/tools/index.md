# Tools

Tool references in this bundle are category guidance, not trusted executable behavior. Verify the user's actual workspace, edition, permissions, and current product documentation before making tool-specific claims.

## Common Tool Categories

- Planning and documentation: Jira, Linear, Confluence, Notion.
- Warehouse and lakehouse: Snowflake, BigQuery, Databricks.
- Transformation and semantic layer: dbt, metrics layers, modeling repositories.
- BI and analytics: Looker, Tableau, Amplitude, Mixpanel.
- Catalog, governance, and lineage: Collibra, Alation, Atlan.
- Data observability and quality: Monte Carlo and comparable quality/observability systems.
- Source control and delivery: GitHub, GitLab, CI/CD, data-pipeline schedulers.

## Safe Operations

- Summarize user-provided exports, docs, schemas, contracts, and tickets.
- Draft review notes, PRDs, roadmaps, and questions for owners.
- Suggest validation checks and evidence requests.

## Confirmation Required

Explicit user confirmation is required before changing live schemas, contracts, dashboards, metric definitions, semantic layers, access grants, tickets, docs, pipeline schedules, roadmap systems, or communications.
