# Source and License Notes

This bundle is original guidance synthesized from public taxonomy and practitioner sources. It links and summarizes sources rather than copying proprietary artifacts.

## Public Sources Used

- ESCO ICT product manager occupation: candidate taxonomy adjacency.
- O*NET OnLine role pages: taxonomy context and caution for the candidate-provided O*NET code.
- Practitioner writing from Alation, Monte Carlo, Collibra, DataScience-PM, and Data Products Substack: general context on data products, data product management, data quality, governance, lifecycle, consumers, and data contracts.

## Grounding Limits

No public source defines the user's local schemas, metric logic, data contracts, SLAs, quality thresholds, access rules, or governance process. Bundle outputs should ask for user-provided or inspected source evidence before making those claims.
