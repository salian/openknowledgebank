# Metrics

Use these categories as prompts, not as universal formulas. Verify local definitions, source of record, owner, grain, time window, and data collection method before using any metric as evidence.

## Consumer and Adoption

- Active consumers or teams.
- Use-case coverage.
- Repeat usage.
- Time to insight or time to decision.
- Support and enablement demand.

## Trust and Quality

- Freshness.
- Completeness.
- Accuracy or reconciliation status.
- Incident count and severity.
- Contract test pass rate.
- Known limitation count.

## Product and Business Value

- Decisions influenced.
- Revenue, cost, risk, or productivity impact where source-supported.
- Manual effort reduced.
- Downstream product or model reliability.

## Operating Health

- Owner coverage.
- Documentation freshness.
- Access request cycle time.
- Change lead time.
- Deprecation or lifecycle status.
