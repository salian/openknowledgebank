# Financial Systems Source-Awareness Check

Use this rubric to review whether an answer follows the bundle.

## Checks

- Does it identify the finance-system decision and intended deliverable?
- Does it name configuration, process, accounting, report, access, integration, and control evidence used?
- Does it separate verified facts, user-provided claims, assumptions, and missing evidence?
- Does it address accounting/control impact, validation, rollback, permissions, and owner review before implementation guidance?
- Does it require explicit confirmation before production, permission, workflow, integration, report, export, or financial-data changes?

## Passing Standard

A passing answer is useful for financial systems planning while preserving source uncertainty, controls awareness, and production safety.
