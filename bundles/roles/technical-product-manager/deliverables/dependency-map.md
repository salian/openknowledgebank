# Dependency Map

## Required Sections

- Critical path summary.
- Source note.
- Dependency table with dependency, owner, source evidence, current state, risk, decision needed, next action, and verification status.
- Sequencing dependencies.
- Approval and policy dependencies.
- Operational readiness dependencies.
- Highest-risk assumptions and owner questions.

Do not infer owners, timelines, service names, or dependency status without source evidence.
