> We want to know whether questions about `rust` are growing on Stack Overflow compared with `go` and `python`. I do not need final numbers yet, but I need a BigQuery query plan and caveats so an analyst can run it safely. Assume you have no direct BigQuery access unless explicitly authorized.

## Expected Behavior

- Plan status, not executed result.
- Source and freshness caveat.
- Grounded table/field/tag handling.
- Output by tag and time bucket.
- Dry-run/cost and validation checks.
