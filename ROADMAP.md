# Roadmap

OpenKnowledgeBank is being built as a free, open, inspectable, and remixable library of knowledge bundles for AI agents.

## Phase 0: Foundation

- Seed the repository structure.
- Define bundle organization for roles, industries, capabilities, tools, frameworks, compliance, jurisdictions, deliverables, and datasets.
- Add starter registry files.
- Add contribution and licensing docs.
- Create the first seed bundle skeleton.

## Phase 1: Useful First Release

- Create `ga4-analytics-specialist` as the first full production benchmark bundle.
- Keep `performance-marketer` as a seed skeleton until after the benchmark.
- Add several more bundles, including non-role bundles where useful.
- Add examples that show how to use bundles with agents.
- Add validation tooling for bundle structure and metadata.
- Improve the registry so tools and websites can consume it.
- Publish clear contribution guidance.

## Phase 2: Better Quality Signals

- Add bundle status levels such as draft, review, and stable.
- Add source coverage and validation indicators.
- Add more realistic examples and workflows.
- Add packaging or release automation for downloadable bundles.
- Improve docs for bundle authors and agent builders.
- Add contributor metadata and public contributor profile support.

## Phase 3: Broader Ecosystem

- Support external OpenKnowledge repositories in the registry.
- Add contributor recognition.
- Add reference integrations.
- Grow the catalog around high-value roles, industries, capabilities, tools, frameworks, compliance regimes, jurisdictions, deliverables, and datasets.
